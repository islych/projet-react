# Assets Directory

This directory contains the app assets. For now, we'll use default Expo assets.

To add custom assets:
1. Replace icon.png (1024x1024)
2. Replace splash.png (1284x2778 for iPhone 12 Pro Max)
3. Replace adaptive-icon.png (1024x1024)
4. Replace favicon.png (48x48)

For testing, Expo will use default assets if these files are missing.
